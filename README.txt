Love Calculator
This program is a love calculator that calculates the compatibility score between two names. It determines the strength of the relationship based on the occurrence of specific letters in the names.

Getting Started
To use the Love Calculator script, follow these steps:

1. Make sure you have Python installed on your system.
2. Clone or download the repository.

Prerequite

python 3.x

Usage

Open a terminal or command prompt.

Navigate to the directory where the code file is located.

Run the following command to execute the script:loveCalculator.py

Example

Welcome to the Love Calculator love birds!
Enter the first name: John
Enter the second name: Jane
Score: 75
Based on the calculated score, the program provides an interpretation of the relationship strength.

Interpretation of Scores

Scores below 10 or above 90: "You go together like Coke and Mentos."
Scores between 40 and 50 (inclusive): "You are alright together."
Other scores: No specific interpretation.
